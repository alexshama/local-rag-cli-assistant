PYTHON KNOWLEDGE BASE
FORMAT: QA + DEFINITIONS
====================

[GENERAL]

Вопрос: Что такое Python?
Ответ: Python — интерпретируемый высокоуровневый язык программирования с динамической типизацией и автоматическим управлением памятью.

Вопрос: Что такое интерпретатор Python?
Ответ: Интерпретатор — программа, выполняющая исходный код Python построчно без предварительной компиляции.

Вопрос: Что такое PEP?
Ответ: PEP — документы, описывающие стандарты, соглашения и предложения по развитию Python.

====================

[DATA TYPES]

Вопрос: Какие типы данных существуют в Python?
Ответ: int, float, complex, bool, str, list, tuple, set, frozenset, dict, None.

int — Целочисленный тип данных произвольной точности.
float — Тип данных для представления чисел с плавающей точкой.
complex — Тип данных для комплексных чисел.
bool — Логический тип данных со значениями True и False.
str — Неизменяемая последовательность символов.
list — Изменяемая упорядоченная коллекция элементов.
tuple — Неизменяемая упорядоченная коллекция элементов.
set — Неупорядоченная коллекция уникальных элементов.
frozenset — Неизменяемая версия множества set.
dict — Коллекция пар ключ–значение.
None — Специальное значение, обозначающее отсутствие значения.

Вопрос: Чем отличаются изменяемые и неизменяемые типы данных?
Ответ: Изменяемые объекты можно менять после создания, неизменяемые — нельзя.

Изменяемый объект — Объект, состояние которого можно изменить после создания.
Неизменяемый объект — Объект, значение которого нельзя изменить после создания.

====================

[VARIABLES AND MEMORY]

Вопрос: Что такое переменная в Python?
Ответ: Переменная — это имя, которое ссылается на объект в памяти.

Переменная — Имя, ссылающееся на объект в памяти.
Объект — Сущность в памяти, имеющая тип, идентификатор и значение.
Ссылка — Указатель на объект в памяти.

Вопрос: Что делает функция id()?
Ответ: Функция id() возвращает уникальный идентификатор объекта.

id() — Функция, возвращающая уникальный идентификатор объекта.

====================

[OPERATORS]

Вопрос: Какие операторы существуют в Python?
Ответ: Арифметические, логические, сравнения, присваивания, побитовые.

Оператор — Символ или ключевое слово, выполняющее операцию над операндами.
Арифметические операторы — Операторы для выполнения математических вычислений.
Операторы сравнения — Операторы для сравнения значений.
Логические операторы — Операторы and, or, not.
Операторы присваивания — Операторы связывания имени с объектом.
Побитовые операторы — Операторы для работы с битами.

====================

[CONTROL FLOW]

Вопрос: Какие управляющие конструкции есть в Python?
Ответ: Условные операторы, циклы и управляющие операторы перехода.

if — Условный оператор.
elif — Дополнительная ветка условия.
else — Альтернативная ветка условия.
for — Цикл перебора элементов.
while — Цикл с условием.
break — Немедленный выход из цикла.
continue — Переход к следующей итерации.
pass — Пустой оператор.
match — Структурное сопоставление с образцом.

====================

[FUNCTIONS]

Вопрос: Что такое функция в Python?
Ответ: Функция — именованный блок кода, принимающий аргументы и возвращающий значение.

Функция — Именованный блок кода.
def — Ключевое слово определения функции.
return — Возврат значения из функции.

Вопрос: Чем отличаются аргументы и параметры?
Ответ: Параметры определяются при объявлении функции, аргументы передаются при вызове.

Аргумент — Значение, передаваемое функции.
Параметр — Переменная, принимающая аргумент.

====================

[SCOPE]

Вопрос: Как работает область видимости в Python?
Ответ: Используется правило LEGB.

Область видимости — Контекст доступности имени.
LEGB — Local, Enclosing, Global, Built-in.
global — Доступ к глобальной переменной.
nonlocal — Доступ к переменной внешней функции.

====================

[EXCEPTIONS]

Вопрос: Что такое исключение?
Ответ: Исключение — ошибка времени выполнения.

Исключение — Ошибка выполнения программы.
try — Блок перехвата исключений.
except — Блок обработки исключений.
finally — Блок, выполняемый всегда.
raise — Генерация исключения.
Exception — Базовый класс исключений.

====================

[OOP]

Вопрос: Что такое класс в Python?
Ответ: Класс — шаблон для создания объектов.

Класс — Шаблон для объектов.
Объект класса — Экземпляр класса.
Метод — Функция внутри класса.
self — Ссылка на текущий объект.
__init__ — Конструктор класса.

====================

[ITERATORS AND GENERATORS]

Вопрос: В чём разница между итератором и генератором?
Ответ: Генератор — упрощённый способ создания итератора через yield.

Итератор — Объект последовательного перебора.
Итерируемый объект — Объект, поддерживающий перебор.
Генератор — Функция с yield.
yield — Возврат значения без завершения функции.

====================

[ASYNC]

Вопрос: Что такое асинхронность в Python?
Ответ: Асинхронность позволяет выполнять операции без блокировки.

async — Объявление асинхронной функции.
await — Ожидание результата.
Coroutine — Асинхронный объект выполнения.
asyncio — Модуль асинхронного программирования.
